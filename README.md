## Sales and Profit Analysis Project
### Project Overview

This project consists of a thorough analysis of data to uncover key insights into sales performance and profit generated from a fictional business-to-business that sells classic models of vehicles such as cars, trains and motorcycles. The dataset covers the years 2003 to 2005. The aim of the project is to inform decision-making, improve commercial success and provide valuable input for shaping the companyâ€™s business strategy.
